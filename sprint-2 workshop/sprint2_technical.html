Meow 1 - {Display:inline;} vs {Display: inline-block;} vs {Display: block;}!!!

  • Inline Elements

    • Inline elements don’t start on a new line, they appear on the same line as the content and tags beside them.

  • Inline-block Elements

    • Inline-block elements are similar to inline elements, except they can have padding and margins added on all four sides.

  • Block Elements

    • A block element always starts on a new line,does not add a line-break after the element, so the element can sit next to other elements.
    And fills up the horizontal space left and right on the web page. You can add margins and padding on all four sides of any block element — top, right, left, and bottom.
