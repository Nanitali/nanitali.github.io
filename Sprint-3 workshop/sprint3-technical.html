What is Function?

We can group a block of code together and associate it with one task.

  1. A Function is a subprogram designed to perform a particular task.
  2. Function are executed when they are called. This is known as invoking a function.
  3. Values can be passed into Function and used within the function.
  4. Function always return a value. In JavaScript, if no return value is specified, the function will return undefined.
  5. Function are objects.

DOM

The Document Object Model (DOM) is an interface to web pages. It defines the logical structure of documents and the way a document is accessed and manipulated,allowing programs to read and manipulate the page's content, structure, and styles.

Why Dom is used in JavaScript?

The DOM represents the document as nodes and objects. ... But it is the same document in both cases. The Document Object Model (DOM) represents that same document so it can be manipulated. The DOM is an object-oriented representation of the web page, which can be modified with a scripting language such as JavaScript.
